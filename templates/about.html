About Webapp

  • This Webapp evaluates your uploaded model on existing dataset for Pneumonia detection.
  • It also evaluates your model on RED attack (black box attack) and FGSM attack (white box attack).
  • It also suggest possible augmentation for training images to increase accuracy of uploaded model.
  • Task remaining for modifying model and showing increase in accuracy after suggested augmentation.

Please note:

  • You can upload only zip file which will have your model.
  • Model must be trained on images of size [height, width] = [224, 224] and
    images must be in RGB format (if B&W then change to RGB format).
  • Model must have class mode as binary and should predict either 0(NORMAL) or 1(PNEUMONIA).
